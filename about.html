about 4Dsoko

  • 4Dsoko is a 4-dimensional sokoban like puzzle game.
    please move the all boxes into the goals.
  • this game is implemented with JavaScript and HTML5 Canvas.
    please enable JavaScript.
  • With IE, excanvas.js is working. please enable ActiveX controls.
